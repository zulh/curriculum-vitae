# Live CV — Ahmad Zulhilmi Ghazali — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build an animated single-page CV website for Ahmad Zulhilmi Ghazali with sticky nav, scroll reveals, typing hero, animated timeline and skill bars, in a navy & sky blue palette.

**Architecture:** Data-driven — all CV content in `src/data/cv.js`, passed top-down via `App.jsx` as props to purely presentational section components. A shared `useScrollReveal` hook provides consistent scroll-triggered animation across all sections.

**Tech Stack:** React 18 + Vite, Tailwind CSS, Framer Motion, react-type-animation, tsparticles, Vitest + React Testing Library

- [ ] **Step 3: Create `src/hooks/useScrollReveal.js`**

```js
import { useEffect } from 'react'
import { useAnimation } from 'framer-motion'
import { useInView } from 'framer-motion'
import { useRef } from 'react'

export const fadeUp = {
  hidden: { opacity: 0, y: 40 },
  visible: {
    opacity: 1,
    y: 0,
    transition: { duration: 0.6, ease: 'easeOut' },
  },
}

export const staggerContainer = {
  hidden: {},
  visible: {
    transition: { staggerChildren: 0.15 },
  },
}

export function useScrollReveal(threshold = 0.15) {
  const ref = useRef(null)
  const controls = useAnimation()
  const inView = useInView(ref, { once: true, amount: threshold })

  useEffect(() => {
    if (inView) controls.start('visible')
  }, [inView, controls])

  return { ref, controls, fadeUp, staggerContainer }
}
```

```jsx
import { useState, useEffect } from 'react'
import { motion } from 'framer-motion'

export default function Navbar({ name, sections }) {
  const [scrolled, setScrolled] = useState(false)
  const [active, setActive] = useState('')

  useEffect(() => {
    const onScroll = () => {
      setScrolled(window.scrollY > 20)

      const current = sections.find(s => {
        const el = document.getElementById(s.toLowerCase())
        if (!el) return false
        const { top, bottom } = el.getBoundingClientRect()
        return top <= 100 && bottom > 100
      })
      if (current) setActive(current)
    }

    window.addEventListener('scroll', onScroll, { passive: true })
    return () => window.removeEventListener('scroll', onScroll)
  }, [sections])

  return (
    <motion.nav
      className={`fixed top-0 left-0 right-0 z-50 transition-all duration-300 ${
        scrolled ? 'bg-primary/90 backdrop-blur-md shadow-lg' : 'bg-transparent'
      }`}
      initial={{ y: -80 }}
      animate={{ y: 0 }}
      transition={{ duration: 0.5 }}
    >
      <div className="max-w-6xl mx-auto px-6 py-4 flex items-center justify-between">
        <span className="font-bold text-accent text-sm tracking-widest uppercase">
          {name}
        </span>
        <ul className="hidden md:flex gap-6">
          {sections.map(s => (
            <li key={s}>
              <a
                href={`#${s.toLowerCase()}`}
                className={`text-sm font-medium transition-colors duration-200 ${
                  active === s ? 'text-accent' : 'text-text-muted hover:text-text-primary'
                }`}
              >
                {s}
              </a>
            </li>
          ))}
        </ul>
      </div>
    </motion.nav>
  )
}
```
